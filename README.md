# La Preuve docs

Ejecutar los siguientes comandos para correr el proyecto:

```shell
    $ npm install
    $ npm start
```

**Requisitos: (Nodejs. + NPM)*

Para buildear proyecto

```shell
    $ npm run build
````

Cada push a master deploya y se puede ver en la siguiente url:
## https://pabmen-test-maquetado.netlify.app/
